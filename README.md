# My Portfolio

## Tech Stack
- HTML/CSS
- Tailwind,
- JavaScript (ES6)
- React + Vite
- Vercel for deployment

### Installation:
1. Clone the Repository
    - git clone https://github.com/Vikas1798/vikasportfolio.git

2. Navigate to the Cloned Repository
    - cd your-repo-name

3. Install Dependencies
    - npm install

4. Start the Project
    - npm run dev

### Live Demo
    https://vikas-hk-portfolio.vercel.app/
